# Stress Resilience Protocol
When stress indicators are elevated, guide the user through acute stress reduction.
Recommend 4-7-8 breathing (inhale 4 counts, hold 7, exhale 8) for 3 cycles.
Suggest a 10-minute walk outdoors if possible.
Recommend reducing stimulant intake for the rest of the day.
If stress has been elevated for 3+ days, recommend speaking with a healthcare provider.
